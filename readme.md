# 学习旅程

这是一个个人学习知识库，用于系统性地记录和管理我的学习过程。

## 目录结构

```
learning-journey/
├── .ai-context/          # AI 辅助学习工具
│   ├── knowledgemap/     # 知识图谱
│   ├── prompts/          # AI 提示词
│   ├── learning-goals.md # 学习目标
│   └── progress-tracker.md # 进度跟踪
├── knowledge-base/       # 知识库文档
├── practice/             # 练习和实战
├── project/              # 项目实践
├── reading-notes/        # 读书笔记
│   ├── books/            # 书籍笔记
│   └── papers/           # 论文笔记
└── weekly-reviews/       # 每周回顾
```

## 学习流程

本仓库遵循 **学习 → 练习 → 项目 → 回顾** 的循环：

1. **知识学习** - 在 `knowledge-base/` 中记录学习到的新知识
2. **实践练习** - 在 `practice/` 中完成习题和练习
3. **项目实践** - 在 `project/` 中做项目巩固知识
4. **回顾总结** - 在 `weekly-reviews/` 中进行周回顾

## 使用说明

- 使用 `.ai-context/` 中的知识图谱和提示词来辅助学习
- 每完成一个知识点或项目，记得更新 `progress-tracker.md`
- 定期进行周回顾，反思学习效果和调整计划

## 开始学习

查看 `learning-goals.md` 了解当前的学习目标和计划。
