# CLAUDE.md

此文件为 Claude Code (claude.ai/code) 在此仓库中工作时提供指导。

## 仓库目的

这是一个个人学习旅程仓库，用于组织和追踪学习进度。仓库采用结构化的方式记录知识、练习、项目、读书笔记和每周回顾。

## 目录结构

```
learning-journey/
├── .ai-context/
│   ├── knowledgemap/     # 知识图谱文件
│   ├── prompts/          # AI 提示词和模板
│   ├── learning-goals.md # 学习目标和计划
│   └── progress-tracker.md # 进度跟踪
├── knowledge-base/       # 结构化的知识文档
├── practice/             # 练习题和问题
├── project/              # 学习项目和实现
├── reading-notes/
│   ├── books/            # 读书笔记和总结
│   └── papers/           # 论文笔记
├── weekly-reviews/       # 每周回顾和反思
└── readme.md             # 仓库概述
```

## 使用此仓库

这是一个基于文档的学习仓库：

- 内容按学习阶段组织（知识 → 练习 → 项目 → 回顾）
- 大多数文件使用 Markdown 格式
- 添加新内容时，根据内容类型放入对应目录：
  - 通用知识和概念 → `knowledge-base/`
  - 代码练习和问题解决 → `practice/`
  - 完整实现和项目 → `project/`
  - 书籍或论文总结 → `reading-notes/books/` 或 `reading-notes/papers/`
  - 每周进度和反思 → `weekly-reviews/`

## 无构建系统

此仓库没有构建系统、测试或部署配置。它是一个个人知识组织系统。
