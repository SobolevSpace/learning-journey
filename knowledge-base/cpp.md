# C++

## 静态变量

### 局部静态变量

局部静态变量的初始化是在**第一次使用时**进行的。

```cpp
void example() {
    static int count = 0;  // 第一次调用时初始化
    count++;
    std::cout << count << std::endl;
}
```

### 特性

1. **延迟初始化**：不进入函数则不初始化，节省资源
2. **线程安全（C++11+）**：保证局部静态变量的初始化是线程安全的
3. **仅初始化一次**：后续调用跳过初始化步骤
4. **生命周期**：在整个程序运行期间存在

### 应用场景

- 单例模式实现
- 缓存/记忆化
- 资源管理的延迟初始化
